# Lab 10 - Loading Strategies

## Objective

Implement all Entity Framework Core loading strategies.

## Implemented

- Eager Loading (Include)
- Explicit Loading (Entry + LoadAsync)
- Lazy Loading (Microsoft.EntityFrameworkCore.Proxies)

## Concepts Covered

- Include()
- Entry()
- Reference()
- LoadAsync()
- Lazy Loading Proxies
- Navigation Properties

## Result

Successfully demonstrated all three loading strategies using EF Core 8 with SQLite.
