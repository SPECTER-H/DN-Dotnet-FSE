# E-Commerce Platform Search Function

Exercise: Implement Linear Search and Binary Search for searching products in an e-commerce platform.

## Files

- `Product.cs` - Product model.
- `SearchOperations.cs` - Linear and Binary Search implementations.
- `Program.cs` - Demonstrates both search algorithms.

## How to build and run

```bash
dotnet build
dotnet run
```

## Sample Output

```text
Linear Search:
104 - Watch (Accessories)

Binary Search:
104 - Watch (Accessories)
```

## Time Complexity

- Linear Search : O(n)
- Binary Search : O(log n)

## Notes

- Binary Search requires the array to be sorted.
- Demonstrates searching techniques commonly used in applications.
